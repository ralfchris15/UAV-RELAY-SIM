# UAV Airborne Relay Simulation

This project is a small interactive visualization demonstrating how
airborne relay UAVs can extend command-and-control beyond terrain-limited
line-of-sight.

The simulator illustrates:

• LOS vs BLOS operations
• Relay-assisted datalink architecture
• Lost-link contingency behavior
• Automatic return to reacquisition waypoint

This is not a flight model or RF simulation. It is intended as a
conceptual demonstration for operational and architectural discussion.
